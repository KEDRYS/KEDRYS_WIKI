Namespace
Только строчные латинские буквы a–z, цифры 0–9, _ и -. Пробелы и CamelCase — нельзя. Вызов: /function aerhold:cool_effect

Noise Router и Density Functions Advanced

Ключевые узлы роутера

  • barrier — граница твёрдого блока
  • fluid_level_flooded — вода в пещерах
  • continents — суша/океан
  • erosion — плоские/горные зоны
  • ridges — хребты и пики
  • final_density — финальная плотность

Типы Density Functions

  • minecraft:noise — Perlin/Simplex шум
  • minecraft:shifted_noise — шум для рек
  • minecraft:interpolated — интерполяция
  • minecraft:spline — кривые рельефа
  • minecraft:add, minecraft:mul
  • minecraft:min, minecraft:max
{
  "type": "minecraft:interpolated",
  "argument": {
    "type": "minecraft:shifted_noise",
    "noise": "minecraft:clay_bands_offset",
    "xz_scale": 1.5, "y_scale": 0.0,
    "shift_x": "minecraft:island_noise",
    "shift_y": 0, "shift_z": "minecraft:island_noise"
  }
}

Splines: кривые фильтрации рельефа

Переводят «значение шума» в «высоту горы». derivative — скорость изменения: высокое = крутой склон.

"points": [
  { "location": -1.0, "value": 0.1, "derivative": 0.0 },
  { "location":  0.0, "value": 0.5, "derivative": 1.2 },
  { "location":  1.0, "value": 1.2, "derivative": 0.0 }
]

Раздел V Scoreboard: переменные и кулдауны

Создание и кулдаун

# load.mcfunction
scoreboard objectives add cooldown_timer dummy
scoreboard objectives add mana_points dummy

# tick.mcfunction — уменьшаем КД
scoreboard players remove @a[scores={cooldown_timer=1..}] cooldown_timer 1

# Сработать если КД = 0, поставить 100 тиков (5 сек)
execute as @a[scores={cooldown_timer=0},nbt={SelectedItem:{id:"minecraft:iron_sword"}}] at @s run summon lightning_bolt ~ ~ ~
execute as @a[scores={cooldown_timer=0},nbt={SelectedItem:{id:"minecraft:iron_sword"}}] run scoreboard players set @s cooldown_timer 100
Запомни
1 сек = 20 тиков. КД 5 сек = 100, 10 сек = 200, 1 мин = 1200.

Операции скорборда

OpПримерСмысл

+=operation @s coins += @p rewardСложить
-=operation @s hp -= @e[limit=1] dmgВычесть
*=operation @s val *= #const twoУмножить
/=operation @s pct /= #const maxРазделить

title @a actionbar ["",{"text":"КД: ","color":"gray"},{"score":{"name":"@s","objective":"cooldown_timer"},"color":"red"}]

# Визуальный кулдаун на предмете (1.21+)
item cooldown @s minecraft:iron_sword 100
Раздел VI Command Storage: база данных

Storage vs Scoreboard

Scoreboard — числа

  • Только целые числа
  • Быстрые вычисления
  • Таймеры, счётчики, HP

Storage — JSON-объекты

  • Строки, массивы, объекты
  • Не исчезает после выхода
  • Квесты, инвентарь, RPG
# Записать данные квеста
data modify storage aerhold:players CurrentQuest set value {id:"kill_frost_giant",progress:0}

# Проверить активен ли квест
execute if data storage aerhold:players {CurrentQuest:{id:"kill_frost_giant"}} run say Квест активен!

# Профиль персонажа
data modify storage aerhold:char Profile set value {name:"KERDYS",class:"berserker",kills:0}

Раздел VIII Raycasting: магические лучи

Алгоритм рекурсивного луча

# ray/start.mcfunction
summon marker ~ ~1.6 ~ {Tags:["aerhold_ray","new_ray"]}
execute as @e[tag=new_ray] at @s run function aerhold:ray/travel
tag @e[tag=new_ray] remove new_ray

# ray/travel.mcfunction
tp @s ^ ^ ^0.5
execute as @s at @s if entity @e[type=!player,type=!marker,distance=..1.2] run function aerhold:ray/hit
execute as @s at @s if block ~ ~ ~ air run function aerhold:ray/travel
execute as @s at @s unless block ~ ~ ~ air run kill @s

# ray/hit.mcfunction
damage @e[type=!player,type=!marker,distance=..1.2,limit=1] 8 aerhold:frost
particle minecraft:snowflake ~ ~ ~ 0.3 0.3 0.3 0.05 30
kill @s

Раздел XXI Оптимизация и автоматизация

Советы по оптимизации

  • Используй теги #c:iron_ingots вместо прямых ID в рецептах — совместимость с модами
  • Никаких тяжёлых вычислений в tick — только диспетчер с условиями
  • Звуки: формат .ogg, битрейт ≤128 kbps
  • Atlas в assets/minecraft/atlases/blocks.json — все текстуры одной картой
  • Schedule вместо тик-счётчиков для долгих таймеров
# Python-генератор 500 рецептов
import json, os
for metal in ["iron","gold","netherite","diamond"]:
    for weapon in ["sword","axe","pickaxe"]:
        recipe = {"type":"minecraft:crafting_shaped","pattern":["XX","X "," X"],
                  "key":{"X":{"tag":f"c:{metal}_ingots"}},
                  "result":{"id":f"aerhold:{metal}_{weapon}","count":1}}
        path = f"data/aerhold/recipes/{metal}_{weapon}.json"
        os.makedirs(os.path.dirname(path), exist_ok=True)
        with open(path,"w") as f: json.dump(recipe,f,indent=2)
Раздел XXII Макросы: динамические функции

Три способа передачи аргументов 1.20.2+

1. Литерально

function aerhold:give_item {
  item: "minecraft:diamond_sword",
  name: "Меч Одина", color: "gold"
}

# give_item.mcfunction
$give @s $(item)
$tellraw @s {"text":"$(name)","color":"$(color)"}

2. Из Storage

data modify storage aerhold:macro TpData set value {x:100,y:64,z:-200}
function aerhold:teleport with storage aerhold:macro TpData

# teleport.mcfunction
$tp @s $(x) $(y) $(z)

3. Координаты игрока → телепорт домой

# Сохраняем позицию
execute store result storage aerhold:waypoints HomeFlat.x double 1 run data get entity @s Pos[0]

Раздел XXXIII Schedule: отложенные функции

Синтаксис и паттерны

КомандаЧто делает
schedule function f:n 20tЧерез 20 тиков (replace)
schedule function f:n 20t appendДобавить в очередь
schedule function f:n 1sЧерез 1 секунду
schedule clear f:nОтменить
# Ритуал призыва — цепочка событий
particle minecraft:soul_fire_flame ~ ~1 ~ 1 1 1 0.05 50
playsound minecraft:block.respawn_anchor.charge ambient @a ~ ~ ~ 2 0.5
schedule function aerhold:ritual/step_2 40t
# step_2: дракон + тайтл + schedule step_3 60t
# step_3: summon aerhold:ice_giant
append vs replace
replace (дефолт) — новый вызов заменяет старый. append — несколько независимых экземпляров (нужно для нескольких ритуалов одновременно).
